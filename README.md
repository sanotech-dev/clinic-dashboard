# 🏥 Clinic Dashboard

سیستم مدیریت بیماران با طراحی Glassmorphism.
پروژهٔ آموزشی برای تسلط بر JavaScript DOM و LocalStorage.

## ✨ ویژگی‌ها

- ثبت بیمار با اعتبارسنجی فرم
- نمایش آمار لحظه‌ای (کل بیماران / امروز)
- جستجوی زنده (Live Search)
- حذف داینامیک با Event Delegation
- ذخیره‌سازی دائمی با LocalStorage

## 🛠 تکنولوژی‌ها

- HTML5 Semantic
- CSS3 (Glassmorphism, Grid, Flexbox)
- Vanilla JavaScript (ES6+)

## 🚀 اجرا

کافیه فایل `index.html` رو توی مرورگر باز کنی.
نیازی به سرور یا بستهٔ جانبی نیست.
